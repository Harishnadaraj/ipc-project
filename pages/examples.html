Examples

Here are simple and clear examples showing how each IPC mechanism (FIFO, Message Queue, Shared Memory) works in real programs. These examples help during viva and while explaining concepts to faculty.

1. FIFO (Named Pipe) Example

This example shows how a producer sends text to a consumer using a FIFO.

Producer Code:

mkfifo("myfifo", 0666); int fd = open("myfifo", O_WRONLY); write(fd, "Hello from FIFO!", 16); close(fd);

Consumer Code:

int fd = open("myfifo", O_RDONLY); char buffer[100]; read(fd, buffer, sizeof(buffer)); printf("Received: %s\n", buffer); close(fd);

Use Case: Real-time logging from one process to another.

2. Message Queue Example

Message Queues allow prioritized, structured messages.

Sender:

mqd_t mq = mq_open("/mqdemo", O_WRONLY); char msg[] = "Task completed!"; mq_send(mq, msg, sizeof(msg), 5); // Priority = 5 mq_close(mq);

Receiver:

mqd_t mq = mq_open("/mqdemo", O_RDONLY); char buffer[128]; unsigned int prio; mq_receive(mq, buffer, sizeof(buffer), &prio); printf("Received (%u): %s\n", prio, buffer); mq_close(mq);

Use Case: Event-based communication between processes.

3. Shared Memory Example

This example shows two processes sharing data using a memory block + semaphore.

Writer:

int shm = shm_open("/shm", O_CREAT | O_RDWR, 0666); ftruncate(shm, 1024); char* ptr = mmap(0, 1024, PROT_WRITE, MAP_SHARED, shm, 0); sprintf(ptr, "Shared Memory Rocks!");

Reader:

int shm = shm_open("/shm", O_RDONLY, 0666); char* ptr = mmap(0, 1024, PROT_READ, MAP_SHARED, shm, 0); printf("Read: %s\n", ptr);

Use Case: Passing high-speed data (like sensor inputs) between processes.

4. Security Token Example

Security ensures only authorized processes access IPC channels.

Check Token:

if (!validate_token("client.token")) { printf("Unauthorized access denied!\n"); exit(1); }

Use Case: Preventing unknown processes from opening shared memory.
